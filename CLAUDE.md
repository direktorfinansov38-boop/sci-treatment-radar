# SCI Treatment Radar — CLAUDE.md

## Что это за проект

Личный Telegram-бот для мониторинга новостей о лечении травмы спинного мозга (ТСМ).
Владелец проекта живёт с травмой спинного мозга 13 лет — бот нужен ему лично,
чтобы следить за научными разработками которые могут напрямую его касаться.

## Приоритеты мониторинга

1. **Стволовые клетки** — главная тема. Мезенхимальные, нейральные, iPSC, экзосомы.
2. **Российские проекты** — особо важны: Матриселф, НейроМат, Сколково и аналогичные.
3. **Клинические испытания** — любые фазы, особенно first-in-human.
4. **Нейростимуляция, импланты, экзоскелеты** — вторичный приоритет.

## Расписание

Раз в 3 дня, в 10:00 по Улан-Батору (02:00 UTC).
Глубина поиска: 4 дня назад (с перекрытием чтобы ничего не пропустить).

## Формат Telegram-сообщений

Каждая статья — отдельное сообщение:
- Картинка с сайта (og:image) если есть
- Эмодзи источника + флаг страны
- Заголовок на русском (автоперевод через Google Translate)
- Дата и источник
- Короткая ссылка (TinyURL)

Перед статьями — отдельное header-сообщение с датой и количеством материалов.
Если настроен ANTHROPIC_API_KEY — в header включается AI-сводка от Claude.

## Источники данных

- **PubMed** — научные статьи (NCBI eUtils API, бесплатно)
- **ClinicalTrials.gov** — клинические испытания (API v2)
- **Google News RSS** — новости по поисковым запросам из `config/queries.json`

## Как запустить локально

```bash
pip install -r requirements.txt
cp .env.example .env
# Заполни .env: TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID
python -m spinal_digest_agent run-once
```

## Ключевые файлы

- `config/queries.json` — все поисковые запросы и приоритетные ключевые слова
- `spinal_digest_agent/enrichment.py` — OG-картинка, перевод, короткая ссылка
- `spinal_digest_agent/relevance.py` — фильтрация нерелевантного контента
- `spinal_digest_agent/scoring.py` — ранжирование (свежесть + приоритетные слова)
- `state/sent_history.json` — память бота (что уже отправлено, чтобы не дублировать)

## Деплой

GitHub Actions (`.github/workflows/daily-digest.yml`).
Секреты в Settings → Secrets: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `ANTHROPIC_API_KEY` (опционально).

## Чего НЕ делать

- Не добавлять общие новости об инвалидности, мотивации, благотворительности
- Не отслеживать товары (подушки, матрасы, коляски)
- Не присылать одно и то же дважды (state/sent_history.json следит за этим)
